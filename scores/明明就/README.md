# 《明明就（原调独奏版）》图片识谱转 MIDI 交付说明

## 交付结论

已将 4 张钢琴谱图片识别为 37 小节、左右手双轨的 Type 1 MIDI。当前版本完成了机器结构校验和有图像依据的重点修正，尚未由真人从头到尾跟谱试听，因此定位为“识谱初校版”，不宣称 100% 音符准确。

主文件：`明明就-钢琴双轨-识谱版.mid`

## 原谱结构

- 曲名：《明明就（原调独奏版）》
- 作曲：周杰伦
- 调号：A 大调，3 个升号
- 拍号：4/4
- 印刷速度：四分音符 = 60
- 小节：第 1–37 小节；第 37 小节为独立的终止全音符和弦

## 已应用校正

- 第 23 小节左手系统起始调号被 OMR 漏识别为 2 个升号；MIDI 已按图片修正为 A 大调 3 个升号。
- 灰色延音提示和重叠音头可能被识别成完全相同的双音；导出时已按“同轨、同拍、同音高、同时值”去重，最终 MIDI 没有同拍同音高重复触发。
- 第 10 小节左手声部被日志判定超出小节线 1/8 音符；导出时已将末音收至小节线，未删除其起音。
- 采用图片明确印刷的 60 BPM，不使用外部版本推测速度。

## 机器校验结果

- MusicXML：2 个钢琴谱表，每个 37 小节。
- MIDI：`MThd`、Type 1、10080 ticks/beat。
- 轨道：Conductor、Piano - Right Hand、Piano - Left Hand。
- 音符触发数：右手 328、左手 275，共 603。
- 时长：149 秒（约 2:29）。
- 音域：右手 A3–B5，左手 A1–A4。
- 第 37 小节起点和最终和弦均已验证。

## 必须人工试听的位置

Audiveris 日志中的节拍警告映射到全曲为：第 2、4、10、20 小节。

- 第 2、20 小节：导出后均填满 4/4，但日志存在未分配音符，检查灰色延音提示是否被漏掉或重复。
- 第 4 小节：右手导出时值到 3.75 拍，末尾留有一个十六分音符长度的间隙，需跟谱试听。
- 第 10 小节：重点确认左手最后一个 F# 的实际时值。
- 另外试听四个页面衔接处（12→13、20→21、28→29）和第 37 小节终止和弦。
