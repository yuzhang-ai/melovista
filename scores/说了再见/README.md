# 《说了再见》图片识谱转 MIDI

## 交付物

- `说了再见-钢琴双轨-识谱版.mid`：可直接播放或导入 DAW/编谱软件。
- `说了再见.musicxml`：可编辑的中间乐谱，建议用 MuseScore 打开做人工精修。
- `omr/说了再见-清理后四页.omr`：Audiveris 识谱工程，便于精确修改某个小节。
- `convert_to_midi.py`：可重复执行的 MusicXML 转 MIDI 脚本。

`source/` 保留清理后的四页 PNG；识谱运行日志和可由 PNG 重新合成的多页 TIFF 未纳入仓库。

## 已校验结果

- 曲谱：72 小节，4/4 拍。
- 调号：前段 D 大调（2 个升号），后段 E♭ 大调（3 个降号）。
- 速度：起始 ♩=64；第 33 小节 62→60；第 34 小节回到 64；结尾按原图递减为 60→58→56→54。
- MIDI：Type 1，3 轨（全局事件、右手、左手），1156 个 Note On，时长约 4:32。
- 文件头：已验证为标准 `MThd`。

## 人工试听闭环

这是由截图自动识谱生成的初版，不应当作 100% 人工校谱成品。请优先试听以下位置：

1. 第 47 小节。
2. 第 55–60 小节（密集分解和弦）。
3. 第 65–66 小节（32 分音符密集）。
4. 第 70–72 小节（八度线与渐慢结尾）。

一旦发现错音，优先在 MusicXML/OMR 里只修改对应小节，然后重新运行 `convert_to_midi.py`，无需整首重做。
