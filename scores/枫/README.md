# 《枫》图片识谱转 MIDI 交付说明

## 交付结论

已将 3 张钢琴五线谱图片识别为 59 小节、左右手双轨的 Type 1 MIDI。当前版本完成了机器结构校验与重点小节的图像对照修正，尚未由真人从头到尾跟谱试听，因此定位为“识谱初校版”，不宣称 100% 音符准确。

主文件：`枫-钢琴双轨-识谱版.mid`

## 已应用校正

- 保留第 4 小节 `2/4`，第 5 小节恢复 `4/4`。
- Audiveris 未识别第 42–45 小节的 `8va`；MIDI 右手对应小节已整体升高一个八度。
- 第 50 小节转入升 C 大调；MIDI 已强制使用 7 个升号，修正 OMR 在系统交界处漏识别的调号。
- 第 50 小节两个声部被 OMR 加入了错误的前置休止，导致尾音溢出；已移除前置休止并将对应声部前移一拍，保住小节尾音。
- 第 52、56 小节的溢出对象仅为机器补出的尾部休止，MIDI 归一化时已裁到小节线。
- 采用 `65 BPM`。原图没有印刷速度标记，该数值来自公开同名钢琴 MIDI 的元数据，只作为可调整的默认值。

## 机器校验结果

- MusicXML：2 个钢琴谱表，每个 59 小节。
- MIDI：`MThd`、Type 1、10080 ticks/beat 高分辨率时间基准。
- 轨道：Conductor、Piano - Right Hand、Piano - Left Hand。
- 音符触发数：右手 503、左手 569，共 1072。
- 时长：216.923 秒（约 3:37，按 65 BPM 的直拍版本计算）。
- 拍号：`4/4 → 第4小节2/4 → 第5小节4/4`。
- 转调：第 50 小节起升 C 大调。

## 必须人工试听的位置

Audiveris 日志中的页内节拍警告映射到全曲后为：第 49、50、52、56 小节。建议再重点试听：第 4 小节、42–45 小节 8va、第 50 小节转调、三个页面衔接处，以及最后两小节。

公开参考版本标注为 4:27、997 个音符、65 BPM；本次图片识谱结果与其在时长和音符数量上存在差异，因此该公开信息仅用于风险提示，不能代替逐小节人工核对。
