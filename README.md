# 三八度沉浸式键盘钢琴

使用电脑键盘演奏连续三八度的浏览器乐器。页面采用真实钢琴键盘布局，每个琴键同时标注电脑键位和音高；按下琴键后会亮起，并从对应位置升起水中发光气泡。

声音使用 Web Audio API 直接调度，音频先启动、气泡后创建。采样加载并解码到内存后，演奏热路径不再请求网络。

## 启动

```powershell
npm.cmd install
npm.cmd run dev
```

打开终端显示的本地地址，选择音色，点击一次右上角的“加载并启动”，再使用键盘弹奏。

## 沉浸模式

点击演奏区右上方的“进入沉浸模式”，页面会屏蔽无关按键，仅保留 36 个琴键、`Space` 音区切换和左 `Alt` 长短音切换。退出沉浸模式使用页面按钮。支持 Keyboard Lock 的浏览器会同时尝试全屏键盘锁定；系统保留快捷键仍可能由操作系统接管。

## 音色

- 原声：Salamander Grand Piano 原声采样
- 明亮：同一钢琴采样叠加高频搁架滤波
- 小提琴：真实弓弦采样
- 钢丝弦吉他：真实拨弦采样
- 萨克斯：真实管乐采样

原声与明亮钢琴共用一套约 1.2MB 的采样；其余音色仅在第一次选择时按需加载，避免拖慢首次打开。

## 键位

- C4–B4 全音：`· 1 2 3 4 5 6`
- C4–B4 半音：`Tab Q E R T`
- C5–B5 全音：`7 8 9 0 - = Backspace`
- C5–B5 半音：`U I P [ ]`
- C3–B3 全音：`A S D F G H J`
- C3–B3 半音：`Z X V B N`
- `Space`：将最低一组在 C3–B3 与 C2–B2 间切换
- 左侧 `Alt`：在短音与长音之间切换；短音尾音约 2–3 秒，长音尾音最长约 9 秒；右侧 Alt 不响应

页面会显示最近一次键盘事件到音频调度的 JS 耗时、最近 120 次 P95，以及浏览器报告的基础音频延迟。JS 调度耗时不等于扬声器最终出声的端到端延迟，最终听感需要使用内置扬声器或有线耳机人工确认。
