# 更新日志

本项目从 `v0.2.0` 起遵循语义化版本，并为每次公开版本保留 Git Tag。

## [0.9.0] - 2026-07-19

### 欣赏模式

- 新增双语钢琴曲库：圣诞快乐，劳伦斯先生、蒲公英的约定、致爱丽丝、你离开的事实与 Call of Silence。
- 新增 MIDI 自动演奏，支持播放/暂停、上一首/下一首、进度跳转、循环和 `0.75× / 1× / 1.25×` 速度。
- 自动演奏复用现有 Web Audio 采样并采用 25ms 调度循环与 160ms 前瞻窗口；用户键盘仍可同时跟弹。
- 自动音符使用白金色琴键与光尘，用户演奏继续使用当前场景强调色。

### 曲目与授权

- 《致爱丽丝》内置 Mutopia Project 的 Public Domain MIDI，并在曲库保留来源与授权链接。
- 其余四首受版权保护的曲目提供本地授权 MIDI 导入入口；文件仅在浏览器内解析，不上传服务器或加入发布包。

### 工程

- 引入 `@tonejs/midi` 解析标准 MIDI；解析仅在加载曲目或用户导入文件时发生，不进入自由演奏按键热路径。
- 欣赏模式与中文/英文界面完整联动，并补充可访问名称与自动化检查。

## [0.8.0] - 2026-07-19

### 品牌

- 产品名称升级为“乐境 MeloVista”，突出旋律与窗外风景共同构成的沉浸演奏空间。
- 中文界面使用“乐境 MeloVista”，英文界面使用“MeloVista”，替代实验性质较强的 Six Octave Piano Lab。

### 多语言

- 新增中文与英文完整界面，覆盖场景、音色、演奏模式、音频状态、性能信息、琴键辅助说明与无障碍文本。
- 顶部控制条增加语言切换，用户选择保存在当前设备，并同步更新页面语言和标题。
- 支持 `?lang=zh` 与 `?lang=en` 指定语言；没有明确选择时根据设备语言自动判断。

### 保持兼容

- 语言切换不进入 Web Audio 演奏热路径，不改变键盘映射、采样加载、延迟测量、视频循环与沉浸模式白名单。

## [0.7.0] - 2026-07-19

### 新增

- 四个场景接入本地静音循环视频：沧海听风、山湖静语、雨夜伴灯与暮光之城。
- 每个视频拥有独立的纵向构图焦点，让窗台物件自然停在钢琴上沿附近。
- 从视频画面生成四张 16:9 静态 poster，用于首屏、视频加载和播放失败兜底。

### 性能与可访问性

- 页面先显示静态 poster，客户端挂载后仅加载当前场景视频；场景切换不会同时加载四个视频。
- 视频固定静音，不进入 Web Audio 演奏链路；继续保持声音调度优先于光尘生成。
- 系统启用“减少动态效果”时不加载视频，仅使用静态场景。

### 视觉优化

- 暮光之城的音符光尘从原星空紫色调整为与落日一致的暖金橙色。

## [0.6.0] - 2026-07-19

### 新增

- 可视钢琴改为完整连续展示 C1–B6，共六组、72 个可见琴键。
- C1/C6 与 C2/C3 的切换逻辑保持不变，未启用音区仍显示但标记为“待切换”，避免重复发光。
- 音符光尘的横向起点同步扩展为六音区坐标。

### 视觉实验

- 保持 186px 琴体高度和现有标签字号，仅将横向布局从四等分改为六等分，用于验证完整六八度视图是否过密。
- 最小舞台宽度从 1060px 调整为 1260px，窄屏可横向滚动查看完整琴键。

## [0.5.1] - 2026-07-19

### 优化

- 将可切换低音组的默认音区从 C3–B3 调整为 C2–B2，使页面首次打开时左侧连续露出 C1–B2。
- 保留 `Space` 在 C2/C3 之间切换的原有能力。

## [0.5.0] - 2026-07-19

### 新增

- 增加可切换扩展音区，默认 C1–B1；全音映射为 `Insert Home PageUp NumLock / * -`，半音映射为 `Delete End Numpad 7 8 9`。
- 小键盘 `+` 可将扩展音区切换到 C6–B6，并将该组琴键移动到可视钢琴最右侧。
- 增加 C1 与 C6 音区的 Salamander Grand Piano 真实采样锚点，避免依赖跨整八度变调。
- 沉浸模式白名单加入全部 12 个扩展琴键和小键盘 `+`。

### 优化

- 可视钢琴改为四组并排，整体高度从 246px 缩至 186px，释放更多窗台与场景视野。
- 性能信息与光尘起点随琴体同步下移，继续保持音频优先于视觉粒子。

### 保持兼容

- 保留 `Space` C2/C3 切换、左 `Alt` 长短音、五种音色与四个沉浸场景。

## [0.4.0] - 2026-07-19

### 新增

- 在海岸午后之外增加林间晴窗、雨夜公寓与星空露台三个原创沉浸场景。
- 增加玻璃质感场景选择器，切换时同步改变背景、环境光与音符粒子颜色。
- 增加欣赏模式实施方案，覆盖 MIDI 曲库、自动演奏、用户跟弹、授权与验收标准。

### 优化

- 所有音高的光尘统一上升到相同最高处，并增强光点尺寸、拖尾、亮度和持续可见时间。
- 音色选择由浏览器原生下拉框改为与顶部控制条一致的自定义玻璃弹层。
- 新场景在页面空闲后预加载，场景资源使用本地压缩 WebP，不进入音频调度热路径。

### 保持兼容

- 继续保证声音先于视觉粒子启动，并保留三八度键位、五种音色、长短音、低音区切换与沉浸模式白名单。

## [0.3.0] - 2026-07-19

### 新增

- 增加首个场景皮肤“海岸午后”，使用原创阳光海景琴房背景、暖色玻璃控制条和临窗钢琴布局。
- 将水下气泡替换为随音高上升的金色音符光尘，粒子始终在音频启动之后创建。
- 将性能详情收纳到可展开面板，同时保留调度、P95 与当前音的紧凑演奏信息。
- 为场景增加社交分享元数据与本地压缩 WebP 预览图。

### 保持兼容

- 保留三八度键位、五种真实采样音色、`Space` 低音区切换、左 `Alt` 长短音切换及沉浸模式白名单。
- 保留短音约 2–3 秒、长音最长约 9 秒的自然尾音，不改变 Web Audio 低延迟热路径。
