# 更新日志

本项目从 `v0.2.0` 起遵循语义化版本，并为每次公开版本保留 Git Tag。

## [0.2.1] - 2026-07-19

### 修复

- 沉浸模式保留 `Space` 音区切换与左 `Alt` 长短音切换，它们与 36 个琴键一起归入钢琴相关操作。
- 仅屏蔽与钢琴演奏无关的其他按键。

## [0.2.0] - 2026-07-19

### 新增

- 增加沉浸模式入口，开启后仅 36 个钢琴映射键可以触发声音。
- 沉浸模式屏蔽 `Space`、左 `Alt` 及其他非琴键的页面默认行为，减少演奏误触。
- 支持的浏览器会尝试进入全屏并启用 Keyboard Lock；不支持时自动使用页面捕获拦截。
- 页面提供明确的退出按钮和模式状态提示。

### 工程

- 建立版本更新日志与 Git Tag 规范。

## [0.1.4] - 2026-07-19

- 将短音尾音调整为约 2–3 秒的自然衰减。

## [0.1.3] - 2026-07-18

- 将长音改为缓慢自然衰减，尾音最长约 9 秒。

## [0.1.2] - 2026-07-18

- 改为连续三八度真实钢琴键盘界面。
- 增加水中发光气泡动效。
- 增加原声、明亮、小提琴、钢丝弦吉他和萨克斯五种音色。

## [0.1.1] - 2026-07-18

- 接入真实钢琴采样。
- 增加左 `Alt` 长短音切换。

## [0.1.0] - 2026-07-18

- 完成三八度电脑键盘钢琴与低延迟验证页面。
