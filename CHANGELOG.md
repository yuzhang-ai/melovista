# 更新日志

本项目从 `v0.2.0` 起遵循语义化版本，并为每次公开版本保留 Git Tag。

## [0.13.0] - 2026-07-20

### 《花海》曲库

- 根据项目所有者提供的三页完整钢琴谱图片，在本地使用 OMR 识别并生成《花海》双手钢琴 MIDI，不向第三方识谱网站上传谱面。
- 识别范围覆盖引子与第 1–50 小节，补正 A 大调、4/4 拍和 `♩=75`，将三页统一为连续时间轴。
- 生成文件包含左右手两条钢琴轨、787 个音符，总时长约 `2:41.6`，音域为 C♯2–E6，可由现有六音区采样完整播放。
- 曲库新增中英文曲名与作者信息，支持自动演奏、变速、循环、琴键同步和用户同时跟弹。

### 验证

- 自动化检查增加《花海》文件解析、双轨结构、音符数、速度、总时长、音域和曲库入口验证。
- Lint 明确排除本地 `work/` 临时工具目录，避免 Audiveris、MuseScore 等第三方脚本进入项目源码检查范围。
- 保留人工试听闭环；若发现局部识谱偏差，可按具体小节继续精修 MIDI。

## [0.11.3] - 2026-07-19

### 雨夜响度校准

- 使用 EBU R128 实测四条环境音：雨夜约为 `-27.5 LUFS`，比海景与城市场景高约 23–25dB，确认上一版仅使用运行时 `0.42` 系数仍然过响。
- 在素材层将雨夜环境音降低 `27.5dB`，重制后约为 `-55.0 LUFS`，让持续雨声回到钢琴背后的白噪音位置。
- 雨夜环境音更换为 `rain-night-v2.wav`，强制浏览器和 CDN 请求新资源，避免继续复用旧缓存。

### 工程

- 自动化验证增加雨夜版本化资源与 PCM 有效响度比较，防止后续回归为高响度素材。

## [0.11.2] - 2026-07-19

### 乒乓视频循环

- 四段背景视频离线合成为“正放 → 倒放”，循环结束时回到原始第一帧，取消上一版可被察觉的双视频长交叉淡化。
- 重新编码为固定 24fps 的静音 H.264 MP4，并优化首帧关键帧和网络加载顺序；单个文件由原来的约 6–9MB 降至约 3–4MB。
- 视频循环周期扩大为原素材的两倍，倒放发生在素材内部，不依赖浏览器对负播放速度的支持。

### 雨夜音量

- 雨夜场景环境声增加独立 `0.42` 系数，实际输出约为原来的四成；其他三个场景保持现有音量。
- 独立 WAV 无缝循环和场景声音交叉淡化保持不变。

### 工程

- 移除双视频、逐帧边界监测和交叉淡化状态，降低同时解码两路高分辨率视频造成的资源占用。
- 更新自动化验证，检查视频静音、文件体积、乒乓播放结构和雨夜独立音量。

## [0.11.1] - 2026-07-19

### 无缝动态场景

- 单个硬循环视频改为两层视频轮换播放，在循环点提前 `1.25` 秒预播下一层并交叉淡化，隐藏短视频重新开始时的画面跳变。
- 使用逐帧视频回调监测循环边界，旧浏览器回退到动画帧检查；下一层未准备完成时保留原视频重播兜底。
- 两层视频始终静音，场景声音不再依赖 MP4 播放进度。

### 独立环境声

- 从四段场景视频提取原始音轨，采用切半重排和等功率交叉淡化制作四条无缝 WAV 环境音。
- 环境音由 Web Audio 独立解码循环，继续使用 `0.28` 默认音量；切换场景、开启和关闭时均进行 `0.8` 秒淡入淡出。
- 环境声与钢琴声音共用浏览器音频上下文，但不进入钢琴采样、发声和延迟测量链路。

### 工程

- 新增双视频、帧级循环监测、独立环境音资源及 WAV 文件有效性的自动化验证。

## [0.11.0] - 2026-07-19

### 曲库

- 接入项目所有者提供并明确允许发布的《蒲公英的约定》MIDI，内置可直接播放曲目增加至三首。
- 自动演奏速度扩展为 `0.75× / 1× / 1.25× / 1.5× / 2×` 五档，继续保持变速不变调。

### 场景环境声

- 顶部增加双语环境声开关，可独立静音，不影响自由演奏或 MIDI 自动演奏。

### 工程

- 环境声不接入低延迟 Web Audio 演奏链路，避免影响琴键调度。
- 新增第三首内置 MIDI、五档速度和环境声交互的自动化验证。

## [0.10.0] - 2026-07-19

### 通用本地 MIDI

- 欣赏模式增加独立的“导入本地 MIDI”入口，不再要求用户先选择一个预设曲名。
- 导入后自动使用文件名作为曲名，立即启用播放、暂停、进度跳转、循环、变速、琴键同步与跟弹。
- 文件通过浏览器 File API 在当前页面内读取和解析，不发送上传请求；刷新页面后需重新选择。

### 曲库调整

- 内置曲库收敛为两首真正可直接播放的曲目：《致爱丽丝》和《圣诞快乐，劳伦斯先生》。
- 移除仅有名称但无法直接播放的四个占位条目，避免用户误以为曲目已经可用。
- 接入项目所有者提供并明确允许发布的《圣诞快乐，劳伦斯先生》MIDI 文件。

### 工程

- 本地导入曲目使用独立的临时播放槽，不覆盖内置曲目的解析缓存。
- 更新双语界面、页面描述、使用说明与自动化验证。

## [0.9.0] - 2026-07-19

### 欣赏模式

- 新增双语钢琴曲库：圣诞快乐，劳伦斯先生、蒲公英的约定、致爱丽丝、你离开的事实与 Call of Silence。
- 新增 MIDI 自动演奏，支持播放/暂停、上一首/下一首、进度跳转、循环和 `0.75× / 1× / 1.25×` 速度。
- 自动演奏复用现有 Web Audio 采样并采用 25ms 调度循环与 160ms 前瞻窗口；用户键盘仍可同时跟弹。
- 自动音符使用白金色琴键与光尘，用户演奏继续使用当前场景强调色。

### 曲目与授权

- 《致爱丽丝》内置 Mutopia Project 的 Public Domain MIDI，并在曲库保留来源与授权链接。
- 其余四首受版权保护的曲目提供本地授权 MIDI 导入入口；文件仅在浏览器内解析，不上传服务器或加入发布包。

### 工程

- 引入 `@tonejs/midi` 解析标准 MIDI；解析仅在加载曲目或用户导入文件时发生，不进入自由演奏按键热路径。
- 欣赏模式与中文/英文界面完整联动，并补充可访问名称与自动化检查。

## [0.7.0] - 2026-07-19
